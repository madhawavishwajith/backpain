# 🩺 Back Pain Relief & Recovery Solutions

A professional, fully functional, mobile-responsive website for back pain information, treatment options, and recovery solutions. Features an interactive 3D spine model.

## 🌐 Live Demo
👉 [https://madhawavishwajith.github.io/backpain](https://madhawavishwajith.github.io/backpain)

## ✨ Features

- **Stunning Hero Section** - Full-screen hero with animations inspired by modern sports websites
- **Interactive 3D Spine Model** - Rotate and explore the spine in full 3D using Three.js
- **Comprehensive Content** - About, treatments, exercises, prevention, and FAQ sections
- **Smooth Animations** - Powered by Framer Motion for silky-smooth transitions
- **Mobile Responsive** - Fully responsive design that works on all screen sizes
- **Loading Screen** - Professional loading animation on page entry
- **Statistics Section** - Eye-catching number displays
- **FAQ Accordion** - Interactive accordion for frequently asked questions

## 🛠️ Tech Stack

- **Next.js 16** - React framework for production
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Animation library
- **Three.js + React Three Fiber** - 3D model rendering
- **@react-three/drei** - Three.js helpers

## 🚀 Getting Started

Run `npm install` then `npm run dev` and open http://localhost:3000
